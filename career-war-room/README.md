# 应届生求职作战部 · Career War Room

> 7 位 AI 军师 · 30 分钟生成你的《校招作战地图》  

## 这是什么

这是一个 **EasyClaw 标准 Skill 包**，
将 6 位专职 AI Agent 打包成一个可一键加载的求职助手团队。

安装这个 Skill 后，你的 EasyClaw 主 Agent 就拥有了：
- 🔮 自我画像挖掘能力
- 🔭 行业匹配能力
- 🔪 简历重写能力
- 🥊 模拟面试能力
- ⚖️ Offer 决策能力
- 💡 情绪陪跑能力

## 快速安装

详见 `INSTALL.md`。

## 文件结构

```text
career-war-room/
├── SKILL.md          # Skill 主入口（EasyClaw 必需）
├── README.md         # 本文件
├── INSTALL.md        # 安装指南
└── agents/           # 6 位子 Agent 配置
    ├── prism/        # 棱镜 · 自我画像官
    ├── scope/        # 望远镜 · 行业匹配官
    ├── scalpel/      # 手术刀 · 简历医生
    ├── arena/        # 沙盘 · 模拟面试官
    ├── balance/      # 天平 · Offer 决策官
    └── lumen/        # 暖灯 · 情绪陪跑官
```
